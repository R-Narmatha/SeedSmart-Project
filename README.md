## 🌱 SeedSmart - Crop Recommendation App

## Overview

SeedSmart is an Android application that helps farmers and users select the most suitable crops based on soil nutrients and environmental conditions using a machine learning model.

## Features

*  Smart crop recommendation
*  Secure user authentication (Firebase)
*  Personalized user history storage (SQLite)
*  User-friendly Android interface

## Technologies Used

* Java (Android Development)
* Firebase Authentication
* SQLite Database
* Machine Learning Model

## Use Case

Helps farmers make data-driven decisions to improve crop yield and efficiency.
